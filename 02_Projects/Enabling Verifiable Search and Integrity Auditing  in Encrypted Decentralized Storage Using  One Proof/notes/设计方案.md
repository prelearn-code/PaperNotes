#### 介绍

首先定义三个群$G\_1，G\_2， G\_T$,这两个群可能是点或者数据的群, 其中设置两个元素，$a\in G\_1,b\in G\_2 , c\in G\_T$ , 其中$e(a,b)=c$ 是一个函数,将$a, b$映射到群$G\_T$的某个元素$c$。

1. type：表示配对的类型，决定构造椭圆曲线的方法，分为a,d,e,f,g等等。
   * type a: 它是定义在域 $F\_q$ 上的曲线 $y^2 = x^3 + x$。它的特点是对称配对（$G\_1 = G\_2$），速度快，适合初学者和很多应用。_本文应用的就是type A类类型_，$G\_1 \* G\_1 = G\_T$
2. q：元素的上界限，即，上述的群元素数据都是模q的，为q阶循环群。
3. r：表示私钥的范围，也是私钥群的阶，同时也是$G\_1，G\_T$的阶，从中可以选取一个随即元素作为私钥，不过这些都是公共参数，可以直接公开的。
   * 公式：$r = 2^{exp2} + sign1\*2^{exp1} + sign0$
4. h：余因子，表示q/r，表示实际使用的“循环子群 (Cyclic Subgroup)”的元素个数
5. exp2, exp1,sing1,sing0：构造的参数，为了更加方便的计算关于q的计算，用这写参数来表示群的阶q。
   * 表示公式：$r = h\*r - 1$
