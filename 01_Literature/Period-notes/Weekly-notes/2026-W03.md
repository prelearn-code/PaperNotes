# 7. 凯撒
## 题目
加密密文：afZ_r9VYfScOeO_UL^RWUc
格式：flag{ }

## WP
```python
c = "afZ_r9VYfScOeO_UL^RWUc"
flag = ""

for i in range(len(c)):
    # 逻辑：密文ASCII + 5 + 索引
    flag += chr(ord(c[i]) + 5 + i)

print(flag)
```
