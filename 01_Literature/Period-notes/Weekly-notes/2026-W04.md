# 1. Lose_md5
## 题目
```python
import hashlib   
for i in range(32,127):
    for j in range(32,127):
        for k in range(32,127):
            m=hashlib.md5()
            m.update('TASC'+chr(i)+'O3RJMV'+chr(j)+'WDJKX'+chr(k)+'ZM')
            des=m.hexdigest()
            if 'e9032' in des and 'da' in des and '911513' in des:
                print des
```

## WP
通过以题目可以知道，其中对于md的信息，有三个字符不知道，通过遍历得到最终的摘要值，这个摘要值就是最终的flag需要的内容。

# 4. Unencode
## 题目
89FQA9WMD<V1A<V1S83DY.#<W3$Q,2TM]

## WP
使用 binascii 的 uu 解码功能。

## code
```python
import binascii

# 待解码字符串
cipher = '89FQA9WMD<V1A<V1S83DY.#<W3$Q,2TM]'

# 使用 binascii 的 uu 解码功能
# 注意：标准的 uu 格式通常包含 begin/end 头，单行解码可以直接用 a2b_uu
try:
    print(binascii.a2b_uu(cipher).decode())
except Exception as e:
    print(e)
```
