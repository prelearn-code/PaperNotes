# 1. 原理
如果一个有理数 $\frac{p}{q}$ 是实数 $\alpha$ 的一个“极好”的近似值，满足以下不等式：

$$\left| \alpha - \frac{p}{q} \right| < \frac{1}{2q^2}$$

那么，$\frac{p}{q}$ 必然出现在 $\alpha$ 的连分数展开的**渐进分数（Convergents）**列表中。


# 2. 例子

## RSA 维纳攻击 (Wiener's Attack)
- **任务**：已知公钥 $(N, e)$，私钥 $d$ 很小，直接恢复私钥 $d$。
    
- 原理：
    RSA 的解密公式是 $ed - k\phi(n) = 1$。
    两边同除以 $d\phi(n)$，可以推导出一个近似关系：
$$\left| \frac{e}{N} - \frac{k}{d} \right| < \frac{1}{2d^2}$$
    这完全符合勒让德定理的形式！这里 $\alpha = \frac{e}{N}$ 是已知的。
- 条件：
    私钥 $d$ 必须足够小，具体来说要满足：
    $$d < \frac{1}{3} N^{\frac{1}{4}}$$
- **如何做**：对 $\frac{e}{N}$ 进行连分数展开，算出的每一个渐进分数 $\frac{k}{d}$ 都是潜在的私钥候选者。
